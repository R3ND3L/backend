## This is a backend folder for the uni-meet project.


## Technology

This backend is created with:

- CometChat Javascript SDK
- Node.js
- Express.js
- MongoDB
- Body-parser
- Cors
- Dotenv
- Nodemon

Install all the dependencies of the application by running
```js
npm install
```
To  run server
```
npm run prebuild
npm run start
```
